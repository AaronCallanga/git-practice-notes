NOTE:
** git rebase <branch>       for cleaner history (will make it linear) 
** git merge <branch>        to keep branching history (will maintain the graph/branching)


# Rebase feature branch with main
    -> git checkout feature/login
    -> git rebase main

# If conflicts happen, fix them manually, then:
    -> git add .
    -> git rebase --continue

# Abort rebase
    -> git rebase --abort
