staged to unstaged(making file unstaged) 
        -> git restore --staged . or <file>
        -> git reset . or <file>
undo changes in working directory(remove unstaged changes) -> git restore . or <file>
undo last commit w/o discarding the changes -> git reset --soft HEAD~1
undo last commit w/ discarding the changes -> git reset --hard HEAD~1
undo specific commit -> git revert <commit-hash>
undo changes in working directory and revert to latest commit 
                -> git checkout HEAD . or <file>
                -> git checkout . or <file>
